Pixel LED Controller 🎮 using Arduino & SD Card 📸

Features:
🔄 Programmable (USB to UART FT232RL)
📂 Able to read micro SD card
🎛️ Able to control with just 4 data lines
🔌 3.3V – 5V power input
🤖 Able to use as an Arduino board

Main Components:
🧠 ATMEGA328P-AU
🔗 FT232RL
🚀 SN74LVC125APW
